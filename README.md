Smart Soil Monitoring System

An  IoT + AI-based Smart Farming System designed to monitor soil conditions in real-time and assist farmers in making data-driven decisions.


# Features

* Real-time soil temperature monitoring
* Soil moisture detection
* Crop health analysis using AI
* Live data transmission using Raspberry Pi
* Web dashboard (React) for visualization
* Backend API using Node.js

# Installation

1. Clone Repository

```bash
git clone https://github.com/yourusername/smart-soil-monitoring-system.git
cd smart-soil-monitoring-system
```

2. Backend Setup

```bash
cd farm-iot-backend
npm install
node server.js
```
# create another Terminal on VS code 

3. Frontend Setup

```bash
cd farm-dashboard
npm install
npm start
```
